# Data

This project uses synthetic AML transaction data generated by IBM AMLSim.

## How to get the data

Run the download script located in `data/raw/`:
```bash
python data/raw/download_dataset.py
```

This will download the following files into `data/raw/`:
- `accounts.csv`
- `transactions.csv`
- `alerts.csv`



## Directory Structure

```
data/
├── raw/                    # Original, unmodified data files
│   └── download_dataset.py # Script to download AMLSim data
├── interim/                # Intermediate transformations
├── processed/              # Final datasets ready for modeling
└── README.md               # This file
```

## Important

Data files (CSVs) are excluded from version control via `.gitignore`. Each user must run the download script independently.
